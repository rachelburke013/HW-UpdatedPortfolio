Trivia Game

This trivia game only displays a single question until the player selects an answer or the time runs out. Then it moves on to the next question. Timers are deployed through Javascript.

Psychic Game

This application randomly picks a letter, and the user has to guess which letter the application chose in a certain number of attempts. The game features dynamically updated HTML and CSS powered by JavaScript code.

Crystal Collector Game

Another fun and interactive game for a web browser. This application systematically updates the HTML with the the use of the jQuery library.

Gif Generator

Using ajax and jquery, this application creates a button for each string in the movie array. When the user clicks on a button, the page grabs 10 static gifs from the Giphy API and places them on the page. The static gif will animate when it is clicked on.
